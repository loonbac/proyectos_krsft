# Plantilla de Materiales

## Formato Simplificado (v3.0)

La plantilla ahora tiene solo 6 columnas. Los números de ITEM se generan automáticamente.

| Columna | Descripción | Ejemplo |
|---------|-------------|---------|
| CANT | Cantidad del material | 10 |
| UND | Unidad de medida | UND, M, KG, PZA |
| DESCRIPCION | Descripción del material | Tubería PVC |
| DIAMETRO | Diámetro (opcional) | 4" |
| SERIE | Serie (opcional) | S-25 |
| MATERIAL | Tipo de material (opcional) | PVC |

**Nota:** Los IDs se asignan automáticamente al importar (1, 2, 3...).

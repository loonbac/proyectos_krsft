# Plantilla de Materiales

Coloca aquí tu archivo Excel personalizado con el nombre:

**plantilla_materiales.xls**

## Columnas requeridas (en orden):
1. CANT - Cantidad
2. UND - Unidad (UND, M, KG, etc.)
3. DESCRIPCION - Descripción del material
4. DIAMETRO - Diámetro (opcional)
5. SERIE - Serie/Clase (opcional)
6. MATERIAL - Tipo de material (opcional)
7. NORMA_DE_FAB - Norma de fabricación (opcional)

Si no colocas ningún archivo, el sistema generará una plantilla automática.
